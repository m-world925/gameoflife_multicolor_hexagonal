Rules for this Game of Life
ALIVE
(BORN)		 A dead cell becomes alive if it has exactly 3 live neighbours.
SURVIVAL A live cell stays alive if it has 2 or 3 live neighbours.
DEAD A live cell dies if it has fewer than 2 live neighbours (UNDERPOPULATION) or more than 3 live neighbours (OVERPOPULATION).
+Additional Rules If the number of color is the same with the other color on the same cell, then the color which is close to black (#000) is stronger (DOMINANT).
